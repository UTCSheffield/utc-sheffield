UTC Sheffield

Mixcandy

The Y11 at UTC Sheffield wanted to build their own disco lights for their prom. The lights needed to be reactive to the music that was being played, so they made some. They were made by a group of 5 Y11 students in an enrichment class. The aim was to make a set of disco lights which would listen to the music and then output different light patterns in time with the music. It took them working on it in which they eventually were successful.

The group of students worked though this using Mixcandy which would listen to the music and flash lights (Neopixels connected though Raspberry Pi via a Fadecandy) Now however it runs on a Raspberry Pi with it connected to a web interface which would allow the user to select different colour and pattern generations algorithms and even change the code being executed live.

The project uses the Fadecandy server for the Raspberry Pi (https://github.com/scanlime/fadecandy) and HTML5 to listen to audio though web browers access to a microphone. LED visualization frames are computed in JavaScript and sent to Fadecandy daemon over WebSockets.
